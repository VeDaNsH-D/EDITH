# edith frontend

React + Vite UI for uploading X12 files and viewing parsed structure, validation issues, summaries, and AI chat.

## Environment

Copy `.env.example` to `.env`:

- `VITE_API_BASE_URL` (example: `http://localhost:8080/api`)

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
